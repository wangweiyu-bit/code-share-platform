# 🚀 快速参考卡

## 一分钟启动

### Windows 用户
```
双击：web/启动.bat
访问：http://localhost:3000/app.html
```

### 其他系统
```bash
cd web
npm install
npm start
# 访问：http://localhost:3000/app.html
```

## 功能速查

### 用户功能
- 注册账号 → 等待审核 → 登录 → 浏览代码 → 下载代码

### 管理员功能
- 登录 → 管理后台 → 审核用户 / 管理代码

## 权限等级

| 等级 | 英文 | 下载限制 |
|------|------|---------|
| 普通用户 | normal | 1个文件 |
| 会员 | vip | 2个文件 |
| 超级会员 | svip | 无限制 |

---

## 常用操作

### 修改管理员密码
```javascript
// 编辑：web/api/db.js
password: '新密码'
```

### 修改下载限制
```javascript
// 编辑：web/api/codes.js
const maxDownloads = user.level === 'normal' ? 1 : user.level === 'vip' ? 2 : -1;
// 改成你想要的数字
```

### 修改端口
```javascript
// 编辑：web/api/server.js
const PORT = 3000; // 改成其他端口
```

## 常见问题

| 问题 | 解决方案 |
|------|---------|
| 启动失败 | 安装 Node.js |
| 端口占用 | 修改端口 |
| 数据丢失 | 使用真实数据库 |
| 无法访问 | 检查防火墙 |
